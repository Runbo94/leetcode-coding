# 283. Move Zeroes

双指针

```
pointer f = first zero index
pointer c = current index

Find first zero first!

| 1 | 2 | 0 | 0 | 3 | 0 | 4 | 5 |
|---|---|---|---|---|---|---|---|
|   |   | f | c |   |   |   |   |
current is 0: current++

| 1 | 2 | 0 | 0 | 3 | 0 | 4 | 5 |
|---|---|---|---|---|---|---|---|
|   |   | f |   | c |   |   |   |
current is not 0: swap f and c, then f++, c++
| 1 | 2 | 3 | 0 | 0 | 0 | 4 | 5 |
|---|---|---|---|---|---|---|---|
|   |   |   | f |   | c |   |   |

| 1 | 2 | 3 | 0 | 0 | 0 | 4 | 5 |
|---|---|---|---|---|---|---|---|
|   |   |   | f |   |   | c |   |
current is not 0
| 1 | 2 | 3 | 4 | 0 | 0 | 0 | 5 |
|---|---|---|---|---|---|---|---|
|   |   |   |   | f |   |   | c |

| 1 | 2 | 3 | 4 | 5 | 0 | 0 | 0 |
|---|---|---|---|---|---|---|---|
|   |   |   |   |   | f |   |   |
```
